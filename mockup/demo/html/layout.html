零碎的组件布局:

浮动布局
    fl       左浮动
    fr       右浮动
    clearfix 清除浮动

块、内联元素
    dib 块级元素 最直观的表现在元素占一行
    db  内联元素 与块级相反

元素显示隐藏
    d_hide 隐藏
    d_show 显示

元素布局定位 pos: position; s: static; f: fixed; a: absolute; r: relative; i: inherit
    pos_s static 默认值。没有定位，元素出现在正常的流中
    pos_f 固顶定位，相对于浏览器窗口进行定位
    pos_a 绝对定位，相对于 static 定位以外的第一个父元素进行定位
    pos_r 相对定位，相对于其正常位置进行定位
    pos_i 继承定位，从父元素继承 position 属性的值
    定位的位置直接写 style（left、top、right、bottom、z-index）

box-sizing
    box_border  border和padding计算入width之内，其实就是怪异模式
    box_content 默认值 不包括border和padding

box
    box      元素转成box布局
    box_ac   垂直方向上的空间利用，也就是垂直居中
    box_flex 根据父元素的剩余空间分成

订单特殊边框
    mod_bd
